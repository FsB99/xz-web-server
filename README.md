# XZ Web Server
XZ Web Server is a minimal high performance php web server, with fast routing and modules such as: firewall. not a HTTP RFC fully compliance.

# Get Started

``` console
php server.php
```

# Performance

| Product | Performance. RPS |
| --- | --- |
| Go (std http) | ~209.734 |
| XZ (Unix) | ~326.849 |
| XZ (Unix with EV) | ~347.768 |
